# Task 2 — Nobel Public Client

## Что реализовано

- Ktor Client + Clean Architecture
- список лауреатов с публичного Nobel Prize API
- фильтр по году и категории
- экран деталей лауреата
- состояния `Loading / Success / Error`
- поддержка изображения лауреата, если в ответе сервиса найдётся `portraitUrl/image/ссылка на изображение`

## Примечание по данным

Публичный Nobel API часто отдаёт разные вложенные поля для person/org и не всегда содержит фото. Поэтому парсинг сделан устойчивым через `JsonObject` и nullable-поля.
